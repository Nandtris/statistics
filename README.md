  - population variance 
  - = σ²=[Σ(xi-μ)²]/N  
  - =[Σ(xi)²]/N-μ²  
  - =[Σ(xi)²]/N-[Σ(xi)]²/N² 
  - μ~总体平均数

- 样本方差  
  - sample variance = s²=[Σ(xi-x)]/n   
  - x~样本平均数
  - sample variance 通常会低估总体方差
 
- 无偏样本方差 unbised sample variance
  - s²=[Σ(xi-x)]/(n-1)
  - 更好的样本方差公式

- satandard deviation 标准差
  - satandard deviation = sqrt（σ²）
  - 与样本单位一致，可以更好的比较数据
  - 方差（样本单位m)，求出的数据单位m²
